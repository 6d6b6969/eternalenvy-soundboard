EternalEnvy epic stream soundboard

Made by Keith Yong. Github repo here. To add new sounds:
  1. Go to the repo.
  2. Fork
  3. Add a new .mp3 sound to the sounds folder. Lets say you added "team.mp3".
  4. Add a new reference to that sound in the big sounds array in script.js, like this

    { says: 'Team? team?... team TEAM', id: 'team', url: soundFolder + 'team' + extension }

  5. Commit and push, and do a pull request!
